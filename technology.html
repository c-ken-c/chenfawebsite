1. 食品製造業：用水等級與系統設計

設計核心： 食品業用水需符合衛福部「食品良好衛生規範準則 (GHP)」及 HACCP/ISO22000 要求。重點在於微生物控制（總菌落數）、感官指標（色度、異味）及化學指標（硬度、餘氯、重金屬）。

1.1 產業分類與建議水質規範

產業類別 建議水質等級 關鍵指標與設計原理 (Key Parameters)
飲料 (碳酸/果汁)純水 (RO Water)需去除餘氯與有機物以防異味；TDS 控制在 50–100 ppm 以維持口感。
乳製品/冰淇淋高純水 / 無菌水嚴格控制耐熱菌與孢子；管路需能耐受 85°C 以上熱水殺菌。
茶飲/咖啡萃取軟水 / 過濾水硬度需適中 (50–150 ppm)，避免萃取問題或設備結垢。
蛋糕甜點/烘焙軟水pH 與硬度影響麵糰與發酵特性。
雞精/保健飲品醫療級純水 (PW)需低 TOC、低內毒素，符合藥典級別的檢測與控制。
肉品加工冷卻水 / 軟水加工前段可保留低量餘氯 (0.5–1.0 ppm)，後段需去除以符合食品風味與安全。

1.2 關鍵設備與工程設計 (精選 30 項)

  • 多介質過濾器 (MMF)

    用途：去除懸浮物。設計公式：過濾面積 = Q / v (m²)，v 常 10–20 m/hr。

  • 活性碳過濾器 (GAC)

    用途：吸附有機物、去除餘氯。EBCT = V_bed / Q，常需 > 5–10 min。

  • 軟化器 (Ion Exchange)

    用途：去硬度。計算：當量負荷 (eq) 與再生周期決定樹脂容量。

  • 保安過濾 (5µm / 1µm)

    用途：保護下游膜系統，減少顆粒破壞高壓泵。

  • 板式換熱器 (PHE)

    用途：溫度控制，熱負荷 Q = m·Cp·ΔT。

  • 逆滲透 (RO)

    用途：主力除鹽/除微生物。膜面積 A = Q_perm / J_op (L/m²·h)。

  • 納濾 (NF)

    用途：選擇性去除多價離子，保留部分礦物質。

  • 超濾 (UF)

    用途：截留膠體、大分子與細菌 (0.01–0.1 µm)。

  • 脫氣 / 脫氧系統

    用途：減少溶氧/CO₂，保護去離子系統與提升電阻率。

  • UV 254 / 185 nm

    用途：254 nm 滅菌，185 nm 降低 TOC；劑量與接觸時間依流量計算。

  • 臭氧系統

    用途：強氧化與終端消毒。設計依 CT 概念 (濃度×時間)。

  • 0.2 µm 終端濾芯

    用途：無菌灌裝前末端過濾，需細菌挑戰測試。

  • SUS316L 衛生儲槽 (SIP 可行)

    用途：貯存拋光水。規格：內表面 Ra ≤ 0.4 µm。

  • 通氣呼吸濾器 (0.2 µm)

    用途：儲槽通氣過濾，防止外空氣污染。

  • CIP / SIP 系統

    用途：線上清洗與滅菌，設計包含溫度、濃度與時間。

  • 熱水滅菌 85°C

    用途：對於不適合化學品的線路，使用熱水循環滅菌。

  • 靜態混合器

    用途：管內藥劑混合，避免死角與不均勻投藥。

  • 導電度 / TDS 在線監測

    用途：即時檢測 RO 或 DI 產水品質並控制 diversion。

  • TOC 在線儀

    用途：高階飲品或醫療飲料需要低 TOC 的線上監控。

  • 衛生級泵浦與閥門

    用途：選用衛生軸封與隔膜閥，減少汙染風險。

  • 流量計 / 壓力計監控

    用途：保養維護與故障排除的第一線參數。

  • 濾材與濾芯管理系統

    用途：排程更換與庫存管理，避免突發停機。

  • 自動加藥系統

    用途：antiscalant、pH 調整、漂白或還原劑精準投加。

  • 袋式過濾器

    用途：階段性高流量粗過濾。

  • 錳砂過濾器 (地下水)

    用途：去除鐵、錳，避免色度與口感問題。

  • 循環迴路（Loop）

    用途：保持循環速度 ≥ 1.0 m/s，減少生物膜生成。

  • PLC / HMI 自動化

    用途：流程控制、歷史數據記錄、HACCP 支援。

  • 緊急旁路與轉換閥

    用途：維護時維持部分生產不中斷，符合 GMP 要求。

  • 品質稽核取樣口

    用途：易取樣設計、SOP 規範化。

技術深解：衛生配管、死角與表面粗糙度

死角控制 (Dead Leg)： 支管長度 L 與直徑 D 的比值建議 L/D < 2，避免死水區滋生細菌（實務上愈短愈好）。

表面粗糙度 (Ra)： 不鏽鋼接觸面建議 Ra ≤ 0.6 µm（機械拋光），若有更高衛生需求採 Ra ≤ 0.4 µm（電解拋光）。

流速建議： 循環迴路流速建議 1.0–1.5 m/s，可維持紊流 (Re > 4000) 並減少生物膜。

2. 科技業：超純水 (UPW) 等級與系統設計

設計核心： 半導體與面板製程要求極低的離子含量、TOC、粒子與微生物；常以電阻率 (MΩ·cm)、TOC (ppb)、粒子數 (particles/mL) 評估質量，目標可達 18.2 MΩ·cm (超純水)。

2.1 代表性產業（10 類）

半導體晶圓製造、面板 (TFT-LCD/OLED)、CMP 清洗、光罩清洗、LED 製程、太陽能電池、精密光學、醫療感測器、電鍍高純度前處理、研究機構 UPW。

2.2 關鍵設備與工程設計 (精選 30 項)

  • 盤式過濾器 / 自清洗濾

    用途：保護 UF，攔截大粒徑雜質。

  • 超濾 (UF)

    用途：去除膠體、膠狀物與部分微生物，SDI < 3。

  • 雙段 RO (2-Pass RO)

    用途：初級脫鹽 + 二級拋光，配合 pH 調整以去硼。

  • 膜脫氣器 (Membrane Degasser)

    用途：降低 DO/CO₂，提升電阻率穩定性。

  • CEDI / EDI

    用途：連續去離子，無需酸鹼再生，常為 UPW 關鍵段。

  • 185 nm UV (TOC 降解)

    用途：產生 OH• 去除有機物，提升 TOC 表現。

  • 拋光混床 (Polishing Mixed Bed)

    用途：最終達到 18.2 MΩ·cm。

  • 終端微過濾 / UF

    用途：末端粒子與微生物控制。

  • PVDF / PTFE 衛生級配管

    用途：低溶出、易清洗的配管系統。

  • 氮氣封存系統

    用途：儲槽頂部惰化，降低 CO₂、O₂ 溶入。

  • 硼去除樹脂塔

    用途：特殊樹脂處理低濃度硼 (ppb 級) 要求。

  • 精密溫度控制 (PHE)

    用途：水溫影響電阻率，控制 ±0.5°C。

  • VFD 與壓力控制系統

    用途：泵浦與管網恆壓、節能。

  • 線上 TOC 與電阻率儀

    用途：關鍵參數監控並觸發警報。

  • 粒子計數器 (LPC)

    用途：0.05–0.2 µm 粒子監控。

  • 溶氧 (DO) 在線儀

    用途：確保 DO 符合製程需求 (ppb 級)。

  • 回收 RO / 再生處理

    用途：降低廢水量、提升經濟性。

  • 樹脂捕捉器 (Resin Trap)

    用途：防止樹脂顆粒進入後段系統。

  • 防震 / 減振設備

    用途：避免泵浦震動引發微粒污染。

  • EDI 電源與控制器

    用途：優化 EDI 效率與壽命。

  • 內襯 FEP/PFA 儲槽

    用途：避免金屬離子溶出。

  • 弱酸陽樹脂塔 (WAC)

    用途：預除 HCO₃⁻，減少 RO 結垢。

  • 混凝 / 沉澱前處理

    用途：大型系統降低懸浮物。

  • 非氧化性生物抑制劑投加

    用途：防止膜生物污堵且不損害膜材。

  • SBS 還原劑

    用途：去除餘氯保護聚酰胺類膜。

  • 阻垢劑投加

    用途：提高 RO 回收並防止晶體結垢。

  • 潔淨室水質資料整合 (BMS)

    用途：製程數據整合，提升追蹤能力。

  • 粒子 / 微生物趨勢分析模組

    用途：預防性維護與污染源追蹤。

  • 備援策略 (N+1)

    用途：關鍵段採 N+1 設計，確保 24/7 運轉。

  • 實驗室 (ICP/IC/TOC)

    用途：定期驗證水質、釐清微量金屬或離子來源。

技術深解：185 nm UV 對 TOC 的影響

185 nm UV 能直接分解有機分子並產生自由基 (OH•)，但會把有機物轉成小分子離子，短期內可能使電阻率下降。因此 UV 後務必設置拋光混床或 EDI 以去除產生的離子，恢復高電阻率。

3. 廢水處理：CMP、市政、電鍍、食品等

設計核心： 廢水處理重點在於分類（分流）與模組化設計：物化 (去 SS、油脂)、生化 (BOD/COD、氨氮)、高級處理 (去鹽、去毒、回收或濃縮)。

  2. 市政污水
  3. 電鍍廢水（含重金屬）
  4. 食品製程廢水（高 COD / 油脂）
  5. 大型汙水處理廠
  6. 科學園區混合工業廢水

  12. 養殖場廢水
  13. 礦業廢水
  14. 建築工地廢水
  15. 汽機車維修廢水

3.2 核心設備與設計 (30 項)

  • 格柵 / 攔污設備

    用途：擋大顆粒保護下游設備。

  • 調節池 (EQ)

    用途：均化水質與流量，容量 V = Q_avg × T_ret (hr)。

  • 初沈池

    用途：去除較大懸浮固體。

  • 油水分離 / DAF

    用途：去除油脂 / 乳化物，DAF 溶氣壓力與泡徑為設計關鍵。

  • 混凝 / 絮凝

    用途：藥劑投加優化 (Jar Test)。

  • 沉澱 / 澄清

    用途：固液分離，表面負荷為重要指標。

  • 曝氣池 (活性污泥)

    用途：BOD 去除，設計 MLSS, SRT。

  • 缺氧 / 反硝化段

    用途：脫氮。

  • 厭氧處理

    用途：高濃度有機廢水產沼氣。

  • MBR (膜生物反應器)

    用途：高效固液分離，設計通量 (LMH) 為關鍵。

  • MBBR

    用途：補強生物處理能力。

  • 鼓風機 / 曝氣系統

    用途：氧供給為能耗主因。

  • 化學氧化 (Fenton / AOP)

    用途：處理難降解 COD，Fenton 最佳 pH 約 3–4。

  • 活性碳吸附

    用途：去除微量有機物與色度。

  • 離子交換 / 電化學

    用途：特定離子或鹽類回收。

  • 蒸發濃縮 / 結晶 (MVR)

    用途：高鹽廢水濃縮以達到 ZLD。

  • 重金屬沉澱

    用途：化學沉澱去除 Cr, Ni, Cu 等。

  • 污泥脫水 (帶式 / 板框 / 疊螺)

    用途：減量化後運送或燃料化處理。

  • 污泥處理 (厭氧消化)

    用途：產沼氣與減量。

  • 消毒池 (氯 / UV)

    用途：放流前消毒。

  • 監測站 (COD / BOD / SS / pH)

    用途：持續監控與法規回報。

  • 廢水熱回收系統

    用途：回收熱能降低運營成本。

  • 溶氣浮選 (DAF)

    用途：高效去除油脂及細微懸浮物。

  • 脫氮創新系統 (Anammox)

    用途：低能耗脫氮解決方案。

  • 雙極膜電透析 (BP-ED)

    用途：鹽分分離並回收酸鹼。

  • 化學藥劑配注與儲存

    用途：安全管理與自動投加。

  • 緊急貯槽 / 應急系統

    用途：處理突發排放事件。

  • 聲學 / 振動抑制

    用途：減少設備故障與延長壽命。

  • 自動化 SCADA / 遠端監控

    用途：資料蒐集、遠端維運及法規回報。

  • 最終濃排放擴散或地下注入方案

    用途：依環評與地理條件選擇排放方式。

技術深解：Fenton 與 MBR 通量計算

Fenton： 最佳工作 pH 3–4，H2O2 與 Fe2+ 投加比需藉 Jar Test 決定，常見重量比 1:1 到 10:1。反應後提高 pH 進行混凝沉澱。

MBR 通量： 生活污水常設計 15–25 LMH，工業難處理水 10–15 LMH，通量越高膜阻塞風險增，需規劃定期 CIP 與間歇反洗。

4. 海水淡化與苦鹹水處理

設計核心： 海水淡化要克服高滲透壓與結垢。典型海水 (TDS ≈ 35,000 ppm) 滲透壓約 25 bar，SWRO 操作壓力通常 55–70 bar，需強調預處理與能量回收 (ERD)。

4.1 關鍵設備與設計 (10+ 項)

  • 深海取水頭 / 取水口

    設計：避免吸入生物，流速低於 0.15 m/s，避開污染與溫排水。

  • 轉鼓篩 / 篩網

    用途：去除海藻、貝類與漂浮物 (mesh 1–3 mm)。

  • 砂濾 / UF 預處理

    用途：降低 SDI、去除懸浮物，UF 趨勢優於傳統砂濾。

  • 抗結垢 / 抗生物投加系統

    用途：antiscalant 與 biocide 以降低膜汙堵與結垢。

  • 高壓泵 (SWRO)

    用途：提供約 55–70 bar 操作壓力，材質需耐海水腐蝕。

  • SWRO 模組 (膜元素)

    用途：主體脫鹽單元，設計回收率通常 35–50%。

  • 能量回收裝置 (ERD / PX)

    用途：回收濃鹽水壓能，效率可達 90% 以上，顯著降低 kWh/m³。

  • 增壓泵 / 補壓系統

    用途：補償壓降並維持系統壓力。

  • 停機沖洗 / 洗膜系統 (Suck-back)

    用途：停機時用淡水置換膜元件內濃水，避免膜損壞。

  • 濃鹽水處置 (Outfall / Evaporation / Injection)

    用途：依地點選擇擴散排放、蒸發結晶或深井注入。

  • 後處理礦化 (Mineralization)

    用途：使用方解石調整 pH 與礦化口感 (飲用用途)。

  • 在線水質監測 (TDS, pH, Cl)

    用途：確保出水品質與操作安全。

技術深解：能量回收 (ERD) 與硼去除

能量回收： 使用高效 ERD (等壓回收器) 可將海淡能耗從 5–8 kWh/m³ 降至約 3.0–3.5 kWh/m³（視海水鹽度與回收率而定）。

硼 (Boron)： 海水中硼 ~4–5 ppm，RO 對硼的去除率受 pH 影響；若需飲用標準 (<1 ppm)，通常採二級 RO 並在段間提高 pH 使硼離子化以利去除。

5. 跨產業設計原理、計算範例與參考

常用計算公式

項目公式 / 說明
日需水量V_day = Σ (process_flow_i × operating_hours_i)，加安全係數 1.2–1.5。
RO 膜面積A = Q_perm / J_op（Q_perm L/h，J_op L/m²·h）。
泵浦功率P (kW) = ρ·g·Q·H / (1000·η)，Q m³/s, H m。
曝氣氧需求以 BOD 負荷與氧利用效率計算，設計鼓風機容量以 kg O₂/d。
調節池容量V_EQ = Q_avg × T_ret，T_ret 常 4–24 小時。

注意事項（整合）

  • 所有接觸水部分優先採用食品級或低溶出材料（SUS316L、PTFE、PVDF）。
  • 設計注意避免死角、保持可清洗性（CIP / SIP）、並規劃取樣點與稽核紀錄。
  • 線上監測參數（電導、TOC、濁度、粒子）應設定 Alert / Action 值並串接 SCADA。
  • 關鍵設備採 N+1 備援，並建立維護 SOP 與耗材備援量。
  • 法規、標準與客戶需求會影響最終選型，設計前應確認最新法規與工廠內控標準。
